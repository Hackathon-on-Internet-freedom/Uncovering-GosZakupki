# Uncovering-GosZakupki
Вскрывая госзакупки — Мониторинг властей. Мы берем открытые данные на госзакупках и перекладываем их в структуры данных и производим сравнения статей затрат, сравниваем стоимость единицы закупок товара со средней ценой товара (например с Яндекс-Маркета). Возможные пути реализации: бэк - Apache Spark, Apache Nifi. Фронт - Python Dash
